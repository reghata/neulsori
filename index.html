🏠 홈 화면

앱을 실행하면 4개의 메뉴가 나타납니다.

  • 🗣️ 대화: 일상 대화에 필요한 단어들을 음성으로 들려줍니다
  • 📖 읽기 연습: 화면에 보이는 단어를 따라 읽는 연습을 합니다
  • 🎙️ 따라말하기 연습: 소리를 들은 후 따라 말하는 연습을 합니다
  • ❓ 도움말: 이 화면입니다

🗣️ 대화 화면 사용법

단어 듣기

• 단어를 누르면 음성이 나옵니다
• 재생 중인 단어는 초록색으로 표시됩니다

단어 정렬하기

화면 상단의 버튼으로 단어를 정렬할 수 있습니다:

  • 가나다라: 가나다라 순서로 정렬
  • 즐겨찾기: 즐겨찾기한 단어만 보기
  • 자주사용: 많이 사용한 단어 순서로 정렬

즐겨찾기 추가/제거

• 단어 옆의 ⭐ 버튼을 누르면 즐겨찾기에 추가됩니다
• 다시 누르면 즐겨찾기에서 제거됩니다

단어 추가하기

• 화면 우하단의 + 버튼을 누릅니다
• 새로운 단어를 입력하고 추가 버튼을 누릅니다

단어 편집/삭제하기

  1. 화면 우상단의 ✏️ 버튼을 누릅니다
  2. 편집하려는 단어의 ✏️ 버튼을 눌러서 수정합니다
  3. 삭제하려는 단어의 🗑️ 버튼을 눌러서 삭제합니다
  4. 완료 후 화면 우상단의 ✓ 버튼을 누릅니다

📖 읽기 연습 사용법

연습 시작하기

  1. 읽기 연습 메뉴를 누릅니다
  2. 연습할 단어를 선택합니다
  3. 화면에 나타난 단어를 보고 읽습니다

음성 인식 사용하기

  1. 🎤 버튼을 누릅니다
  2. 단어를 또렷하게 말합니다
  3. 발음 정확도가 표시됩니다

🎙️ 따라말하기 연습 사용법

연습 시작하기

  1. 따라말하기 연습 메뉴를 누릅니다
  2. 연습할 단어를 선택합니다

단어 듣고 따라하기

  1. 🔊 단어 듣기 버튼을 눌러서 목표 단어를 들어봅니다
  2. 단어 카드를 누르면 목표 단어를 볼 수 있습니다
  3. 🎤 버튼을 누르고 들은 단어를 따라 말합니다
  4. 발음 정확도가 표시됩니다

⚙️ 설정 사용법

각 화면 우상단의 ⚙️ 버튼을 누르면 설정을 변경할 수 있습니다.

글자 크기 조절

  • 보통: 일반적인 크기
  • 크게: 조금 더 큰 글자
  • 아주 크게: 가장 큰 글자

말하기 속도 조절

  • 보통: 일반적인 말하기 속도
  • 느리게: 조금 더 천천히
  • 아주 느리게: 가장 천천히

음성 선택

  • 첫 번째: 내장된 첫 번째 음성 재생
  • 두 번째: 내장된 두 번째 음성 재생

💡 사용 팁

효과적인 사용 방법

  • 매일 조금씩: 하루 10-15분씩 꾸준히 사용하세요
  • 즐겨찾기 활용: 자주 사용하는 단어는 즐겨찾기에 추가하세요
  • 반복 연습: 같은 단어를 여러 번 연습해보세요
  • 단계별 접근: 쉬운 단어부터 시작해서 점차 어려운 단어로 진행하세요

환자분이 사용하기 어려워할 때

  • 글자 크기를 아주 크게 설정해보세요
  • 말하기 속도를 아주 느리게 설정해보세요
  • 처음에는 연습 기능을 사용하다가 점차 대화 기능을 사용해보세요

마이크 사용 시 주의사항

  • 조용한 환경에서 사용하세요
  • 마이크에서 20-30cm 정도 떨어져서 말하세요
  • 너무 크게 말하지 마시고 또렷하게 발음하세요

🆘 자주 묻는 질문

Q. 앱을 최신 버전으로 업데이트하려면 어떻게 하나요?

A. 늘소리는 새로운 기능이 추가되면 자동으로 업데이트됩니다. 최신 버전을 바로 적용하려면, 앱을 완전히 종료했다가 다시 켜주세요.
(방법: 휴대폰의 '최근 실행 앱' 목록에서 늘소리 앱을 위로 쓸어올려 닫은 후, 다시 실행)

Q. 앱이 느려지거나 이상하게 작동해요.

A. 가장 먼저 앱을 완전히 껐다가 다시 켜보세요. 대부분의 일시적인 문제는 이 방법으로 해결됩니다. 그래도 문제가 계속되면 휴대폰을 재시작하는 것이 도움이 될 수 있습니다.

Q. 음성이 나오지 않아요.

A. 먼저 휴대폰의 미디어 볼륨이 켜져 있는지, 무음 모드가 아닌지 확인해 주세요. 그래도 소리가 나지 않으면 앱을 완전히 껐다가 다시 켜보세요.

Q. 제가 추가한 단어 목록을 백업할 수 있나요?

A. 네, 가능합니다. [설정] 화면에서 '치료사 모드'로 들어간 뒤, [📤 단어 내보내기] 버튼을 누르면 현재 단어 목록을 CSV 파일로 저장하여 안전하게 백업할 수 있습니다.

Q. 아이폰 특정 버전에서 마이크가 안돼요.

A. 가끔 iOS를 특정 버전으로 업데이트한 후 마이크 권한에 일시적인 오류가 생기는 경우가 있습니다. 이는 늘소리 앱의 문제가 아닌 iOS 시스템의 버그일 가능성이 높습니다. 휴대폰을 재시작하거나, [설정] > [개인정보 보호 및 보안] > [마이크] 메뉴를 확인해 보시는 것이 좋습니다.

Q. 단어 가져오기를 했는데 글자가 모두 깨져서 나와요.

A. 파일이 잘못된 형식으로 저장되었기 때문입니다. 엑셀이나 메모장에서 단어 목록 파일을 저장하실 때, 반드시 파일 형식을 'CSV UTF-8' 또는 'TXT (UTF-8 인코딩)'으로 지정하여 저장해야 합니다.

Q. 클라우드 공유 링크의 파일을 다운로드할 수 없어요.

A. 모바일 브라우저에서는 다운로드 버튼이 숨겨져 있는 경우가 많습니다. 브라우저 메뉴에서 '데스크톱 사이트'를 선택하면, 컴퓨터 화면처럼 바뀌면서 상단에 [다운로드] 버튼이 나타납니다.

Q. '다른 이름으로 저장'을 하면 안 되나요?

A. 네, 사용하시면 안 됩니다. 모바일에서 '다른 이름으로 저장' 시 파일이 멋대로 엑셀(.xlsx) 형식으로 변환되어 앱에서 읽을 수 없게 됩니다. 반드시 [다운로드] 버튼을 이용해 원본 그대로 저장해야 합니다.

Q. 다운로드한 파일이 '단어 가져오기' 목록에 안 보여요.

A. 안드로이드 보안 정책 때문입니다. 휴대폰의 '내 파일' 앱을 열어 '다운로드' 폴더에 있는 파일을 'Documents' 폴더로 '이동'시킨 후 다시 시도해 보세요.

Q. 휴대폰 '설정'에 늘소리 앱이 없는데, 마이크 권한은 어디서 바꾸나요?

A. 늘소리는 '웹 앱'이므로, 크롬(Chrome) 브라우저의 설정에서 권한을 관리해야 합니다.
(찾는 방법: 크롬 실행 → 메뉴(⋮) → 설정 → 사이트 설정 → 마이크)

Q. 앱이 어떨 때는 설치되고, 어떨 때는 설치가 안 돼요.

A. 브라우저가 사용자의 관심도를 보고 판단하기 때문입니다. 기다릴 필요 없이 수동으로 설치할 수 있습니다.
(방법: 브라우저 메뉴 → '앱 설치' 또는 '홈 화면에 추가')
